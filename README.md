# 📺 BTECH-TV WORLD ULTRA MAX X

A modern IPTV streaming platform built with HTML, CSS, JavaScript, PHP, and JSON.

---

## Features

- 📺 Live IPTV Streaming
- 🌍 Country Filter
- 🎬 Category Filter
- 🌐 Language Filter
- ⭐ Favorites
- 🕘 Recently Watched
- 🔍 Smart Search
- 📱 Responsive Design
- 🌙 Dark Theme
- ⛶ Fullscreen
- 🖼 Picture in Picture
- 🎲 Shuffle Channels
- ⬇ Download Playlist
- 🔄 Refresh Streams
- ⚡ Fast HLS Player
- 📡 M3U Playlist Support
- 📊 Super Admin Dashboard
- 👥 User Login System
- ⚙ Settings Manager
- 📁 JSON Database
- 🐘 PHP API
- 💾 Local Storage Support

## Installation

1. Copy the project into your web server.

```
htdocs/
```

or

```
www/
```

2. Start Apache.

3. Open your browser.

```
http://localhost/BTECH-TV
```

---

## Requirements

- PHP 8+
- Apache / Nginx
- Modern Browser
- Internet connection for remote IPTV playlists
